# Methodology

## Step 1: Data Loading
- Data loaded using pandas (read_csv, read_excel)
- Initial inspection using head(), info(), and describe()

## Step 2: Data Cleaning
- Standardized column names
- Handled missing values
- Converted data types where necessary
- Removed duplicates if present

## Step 3: Data Validation
- Checked for missing values
- Verified numeric ranges
- Ensured logical consistency in columns

## Step 4: Exploratory Data Analysis
- Aggregations using groupby
- Statistical measures (mean, median, standard deviation)
- Trend and distribution analysis

## Step 5: Visualization
- Bar charts for comparisons
- Line charts for trends
- Scatter plots for relationships
- Histograms for distributions

## Step 6: Insight Generation
- Identified patterns and anomalies
- Translated analysis into actionable business insights
